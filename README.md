# Notelifyy

This is a small angular project that demonstrates the CRUD operations.

## Technologies used: 
  
  -Angular 12
  -Bootstrap 5
  -Spring Boot
  -H2 Database


This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 12.1.4.
